## 基岩（异常货物）版式

![Static Badge](https://img.shields.io/badge/license-MIT-blue?style=flat)
  
这是**基岩（异常货物）**版式主题的GitHub存储库。是用于[异常货物委员会中文分部](https://unusual-cargo-cn.wikidot.com/)及其沙盒站[异常沙盒委员会](https://unusual-cargo-cn-sandbox.wikidot.com/)的默认CSS版式。目前由异常货物委员会中文分部站务组维护。

**基岩（异常货物）**版式基于基岩版式（Bedrock）与异常货物版式，由Chhokmah Lu制作。

使用[jsDelivr](https://www.jsdelivr.com/package/gh/parawatch-wiki-cn/AnonTheme)拉取仓库并通过[JSDMirror](https://cdn.jsdmirror.com/)镜像直接在Wikidot中使用。

pages文件夹下的local.html是一个模仿Wikidot结果的原生HTML网页，可以再本地开发测试时查看版式的大概效果。

测试阶段时，可以继续使用main分支，但正式部署时请在GitHub页面右侧打一个Release，填写对应的版本号，以保证jsDelivr拉取稳定的版本，main分支即使出现bug或开发问题也不会影响正常使用。
